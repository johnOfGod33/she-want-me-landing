04

Suppression partielle des données

Il n'est pas possible de supprimer partiellement vos données sans supprimer votre compte, car votre compte est entièrement anonyme et toutes les données y sont liées.

Toutes les données de session sont automatiquement supprimées dans un délai de 90 jours même sans demande explicite.
